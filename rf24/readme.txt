Cubie (Кубик) - сервер
rf24_1 .. rf24_6 - удаленные модули установенные в комнатах

Информация о установленных модулях хранится в БД MySQL. 
Формат:


Name	String	Имя модуля
ID_Mod			Адрес модуля (Последние две цифры адреса)
Status	Boolean	Статус модуля (досупен или нет)
Comment	String	Описание модуля


При включении, запуске программы, Кубик сканирует доступность модулей. Их статус прописывается в соответствующих полях Status. Для доступных модулей в папке /tmp создается каталог с именем Name.  

В каталогах создаются файлы:
export - команды модулю
import - статус датчиков хранение данных с которых не требует накопления (выключатели)
