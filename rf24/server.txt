Cubie (Кубик) - сервер
rf24_1 .. rf24_6 - удаленные модули установенные в комнатах

Информация о установленных модулях хранится в БД MySQL в таблице Modules. 
Формат:


Name_Mod	String	Имя модуля
ID_Mod	Num(2)		Номер модуля (Последние две цифры адреса)	Ключ
Status	Boolean	Статус модуля (досупен или нет)
Comment	String	Описание модуля


При включении, запуске программы, Кубик сканирует доступность модулей. Их статус прописывается в соответствующих полях Status.   

Для хранения данных которые не требуют накопления (выключатели) и передачи команд модулям, в папке /tmp создается каталог с именем Name_Mod. В каталогах создаются файлы:
export - команды модулю 
import - статус датчиков

Описание сенсоров установленных на модуле заносится в таблицу Sensors.
ID_Sen

Значения с датчиков требующих накопления (температура) записывается в таблицы Name_Type. Для каждого типа своя таблица
